If creating visual artifacts (slides, mocks, throwaway prototypes, etc), copy assets out and create static HTML files for the user to view. If working on production code, read the rules here to become an expert in designing with this brand.

Key design rules to internalize before producing any output:
- Dark navy sidebar (#0f1623) + white main area — no gradients, no shadows
- Inter 400/500 only — no bold, no light weights
- 5-agent color system is immutable: Farmer teal, Logistics dark-teal, Energy amber, Market coral, Regulator purple
- Risk status: HIGH #e53e3e / MEDIUM #dd6b20 / LOW #38a169
- Tabler Icons (outline) only — no emoji in UI
- Border-radius max 8px; badges 4px
- Opacity 38% on waiting cards; full opacity when active

If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts OR production code, depending on the need.
